Angular Add Example

A simple directive for adding code examples based on the actual code and markup! Just add AddExample to your element and you're done:D

A really basic example

Lets start with something really simple, by adding the directive AddExample to our element this is what we'll get.

Note that the directive itself and other angular specific attributes added during compilation are removed as we often want to show the code before compilation. However, in some cases we might want to keep the inputs and attributes added to our example element and by passing [keepInputs]="true" to our directive we can achieve just that.

The less basic reality

A real app or library is usually more complex and most often our code is split in to multiple files (JavaScript, TypeScript, html, css etc.), which is good a good practice for organizing our code but i also poses a problem when we're trying to keep examples up to date. So how do we solve that last problem now that our code is split into multiple files?

The following example loads external sources that are passed to the directive and presents them within tabs. You can override the css styles directly or pass other nav options to leverage bootstraps default styles using [navStyle]="'tabs'". We're also targeting another element using [target]="basicExample" (the element we want to add attach our example to) and by passing a custom class name [customClass]="'card-footer'" we can alter the appearance.

For the sake of demonstrating the concept I've added an example inside another example so the first one is what you want to achieve and the second code block is actually how you'd set it up.
